# findFirstDevice()
## Description
Searches the i2c bus and returns the address of the first MAX1704X device found.

## Parameters
**Optional** `expectedVersion : uint16_t`

## Returns
`adress : uint8_t`

## Example
See the example [Find Device](https://porrey.github.io/max1704x/examples/find-device)

None.
